coco

code coverage for ruby 1.9.2, 1.9.3 and 2.0

  • Works with Rails
  • Console report
  • Simple html report
  • Configurable

Version 0.7

released on 2013-06-19

Usage

gem install coco

Require the coco library at the beginning of your tests

require 'coco'

Usually you do this only once, by putting this line in an spec_helper.rb, or test_helper.rb (or whatever you named it).

Config

Configuration is done via a YAML file.

Sample config for a Rails project

:directories: 
- app
- lib
:excludes:
- spec
- config/initializers
:single_line_report: true
