Coco, code coverage for ruby 1.9

Version 0.6 released on 2011-10-30.

Features

  • Simply require 'coco' from rspec
  • Display filenames covered less than 90% on console
  • Simple html report only for files covered less than 90%
  • Report sources with no tests
  • UTF-8 compliant
  • Configurable with a simple yaml file
  • Colorized console output (*nix only)

Note: I have tested coco on linux, with test/unit and Rspec, in rails and standalone app.

Install Coco

To install coco with rubygem:

gem install coco

See the wiki for more information on using coco.
